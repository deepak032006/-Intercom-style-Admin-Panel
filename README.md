## ✨ Features

- 📱 Fully **responsive** layout (desktop & mobile)
- 💬 Chat interface with:
  - Message threads
  - Typing indicators
  - Emoji picker
- 🧑‍💼 User details & metadata sidebar
- 🎨 Clean UI with **TailwindCSS**
- 🎞️ Smooth animations using **Framer Motion**
- 🔁 Dummy data simulation
- 🧭 Sidebar navigation & search
